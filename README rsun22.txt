README rsun22.perl

-f, --from 
  Name of input file Default : gene_1.fna
-l, --fraglen 
  Length of the fragment Default : 200
-t, --mtemp 
  Melting temperature Default : 60
-p, --primerlen 
  Primer length 15 to 25 Default : 20
--help 
  Help information
Example : perl rsun22project.pl -f gene_1.fna -l 200 -t 56 -p 18
